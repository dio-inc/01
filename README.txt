株式会社DIO サイト（静的・ブラウン薄め版） — 配布パッケージ

同梱ファイル:
- index.html
- styles.css
- assets/ (画像・アイコン)

特徴:
- ヒーローキャッチコピーは「確かな技術で仕上げる、床仕上げのプロフェッショナル。」
- サブコピーも床仕上げ全般に対応する表現に変更
- ブラウン薄めカラーを反映済み（メイン:#D7B894、アクセント:#7A5A3D、背景:#FFFBF6）

使い方:
1) ローカルでプレビュー:
   - 解凍後、index.html をブラウザで開いて確認してください。

2) GitHub Pages に公開する（簡単で無料）:
   - GitHub にリポジトリを作成（例: dio-site-brown）
   - ファイルをアップロードしてコミット
   - リポジトリの Settings > Pages から main branch / root を公開に設定
   - 数分で https://<your-username>.github.io/dio-site-brown/ のように公開されます

3) 連絡フォームについて:
   - 現在は mailto を利用しています（メールソフトが開きます）
   - 自動受付フォームにしたい場合は Formspree / Netlify Forms 等に対応可能

カスタマイズ:
- assets/ 内の hero.jpg と work*.jpg を実際の施工写真に差し替え可能
- styles.css で色やフォントの微調整も可能
